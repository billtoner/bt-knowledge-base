# 7z

High-ratio, multi-format archiver with AES-256 (p7zip).

## Create / extract

```bash
7z a out.7z dir/                        # add (create), recursive by default
7z x out.7z                             # extract, preserving paths
7z x out.7z -o/target                   # extract into a directory
7z l out.7z                             # list contents
```

## Encryption & formats

```bash
7z a -p -mhe=on enc.7z dir/             # AES-256 + encrypt the file list too
7z a out.zip dir/                       # also writes zip / tar / gzip / xz ...
7z t out.7z                             # test archive integrity
```

## Notes

- `-mhe=on` hides file names (encrypts headers); without it, names are visible
- `x` preserves directory structure; `e` extracts flat (junk paths)
