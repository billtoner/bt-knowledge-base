# Infrastructure as Code

Declaratively provisioning and configuring infrastructure.

- [ansible](../../tool-notes/ansible.md) — the whole family: ad-hoc `ansible`, `ansible-playbook`, `ansible-vault`, `ansible-inventory`, `ansible-galaxy`
- [terraform](../../tool-notes/terraform.md) — provision infrastructure as code; plan, apply, state, workspaces
- [terragrunt](../../tool-notes/terragrunt.md) — keep Terraform DRY across envs; remote state, module wiring
- [pulumi](../../tool-notes/pulumi.md) — IaC in real languages; stacks, config, encrypted secrets
- [packer](../../tool-notes/packer.md) — build identical machine/container images from one config
