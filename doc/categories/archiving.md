# Archiving

Archiving tools.

- [tar](../../tool-notes/tar.md) — create/extract archives; the flags you always re-look-up
- [zstd](../../tool-notes/zstd.md) — fast modern compression; great ratios, tunable levels
- [zip](../../tool-notes/zip.md) — ZIP archives for cross-platform interop (zip/unzip)
- [7z](../../tool-notes/7z.md) — high-ratio multi-format archiver with AES-256
