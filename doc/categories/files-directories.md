# Files & Directories

Listing, viewing, and syncing files and directories.

- [bat](../../tool-notes/bat.md) — `cat` with syntax highlighting and paging
- [eza](../../tool-notes/eza.md) — modern `ls` replacement
- [rsync](../../tool-notes/rsync.md) — block-level sync over SSH or locally; resumable transfers, `--link-dest` snapshots, dry-run safety
- [tree](../../tool-notes/tree.md) — recursive directory listing as a tree
- [broot](../../tool-notes/broot.md) — interactive tree navigator; fuzzy search + actions
