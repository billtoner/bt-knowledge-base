# Disk

Disk tools.

- [ncdu](../../tool-notes/ncdu.md) — interactive 'what's eating my disk'; navigable du (dust alt)
- [lsblk](../../tool-notes/lsblk.md) — block devices, partitions, mounts, sizes as a tree
- [df](../../tool-notes/df.md) — filesystem free space (df) and per-directory usage (du)
- [parted](../../tool-notes/parted.md) — partition disks (parted scriptable, fdisk interactive)
