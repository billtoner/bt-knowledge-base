# Markup & Docs

Writing and previewing HTML and Markdown.

- [html-snippets](../../tool-notes/html-snippets.md) — modern boilerplates, classless CSS pages, `<details>`/`<dialog>` patterns, useful meta tags
- [grip](../../tool-notes/grip.md) — render local markdown via GitHub's API and serve on localhost; identical to github.com
- [pandoc](../../tool-notes/pandoc.md) — universal document converter; markdown/HTML/PDF/docx and more
- [glow](../../tool-notes/glow.md) — render markdown in the terminal (pairs with kb show)
- [graphviz](../../tool-notes/graphviz.md) — diagrams as code; render DOT graphs to SVG/PNG
