# Cloud CLIs

Command-line and desktop tools for cloud provider services (AWS, GCP, Azure).

- [NoSQL Workbench](../../tool-notes/nosql-workbench.md) — AWS GUI for designing and querying DynamoDB tables
- [gcloud](../../tool-notes/gcloud.md) — Google Cloud CLI — auth, config, and GCP resources
- [gsutil](../../tool-notes/gsutil.md) — Google Cloud Storage from the CLI (gsutil + gcloud storage)
- [bq](../../tool-notes/bq.md) — BigQuery CLI — query, load, extract, manage datasets
- [aws](../../tool-notes/aws.md) — AWS CLI v2 — profiles, SSO, and every service API
- [az](../../tool-notes/az.md) — Azure CLI — auth, subscriptions, and Azure resources
