# Version Control

Git, plus the tools that make its history and diffs readable.

- [git](../../tool-notes/git.md) — the core tool itself; stash, log, diff, reflog, worktrees, bisect, rebase, safe push
- [delta](../../tool-notes/delta.md) — syntax-highlighted, language-aware git diff pager
- [lazygit](../../tool-notes/lazygit.md) — full-screen TUI for git
- [tig](../../tool-notes/tig.md) — TUI for browsing git history, diff, blame, refs
- [gh](../../tool-notes/gh.md) — GitHub from the terminal; PRs, issues, releases, Actions, API
- [pre-commit](../../tool-notes/pre-commit.md) — manage/run git pre-commit hooks (ruff/pytest/secret scan)
