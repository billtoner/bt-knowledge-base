# Build & Packaging

Build & Packaging tools.

- [make](../../tool-notes/make.md) — the ubiquitous build/task runner; targets, prereqs, parallelism
- [just](../../tool-notes/just.md) — modern command runner; like make, but for tasks
- [uv](../../tool-notes/uv.md) — fast Python project/dependency/env manager
- [go](../../tool-notes/go.md) — the Go toolchain; build, test, run, modules, vet
- [gradle](../../tool-notes/gradle.md) — JVM builds (Gradle + Maven); tasks, deps, wrapper
